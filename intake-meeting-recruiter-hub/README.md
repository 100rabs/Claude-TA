# Intake Meeting — Recruiter Hub

A centralized intake meeting platform that streamlines the recruiter-hiring manager alignment process, ensuring every search kicks off with clarity, structure, and shared expectations.

## Overview

The Recruiter Hub provides a structured framework for intake meetings between recruiters and hiring managers. It captures role requirements, generates standardized summaries, and ensures both sides are aligned before sourcing begins — eliminating miscommunication and wasted effort.

## How It Works

1. **Meeting Setup** — Create a new intake meeting for a role, pulling in the JD and hiring manager details.
2. **Structured Capture** — Walk through a guided intake template covering must-haves, nice-to-haves, deal-breakers, compensation, timeline, and team dynamics.
3. **Alignment Scoring** — The tool scores recruiter-hiring manager alignment and flags gaps or mismatches in expectations.
4. **Summary Generation** — Auto-generates a standardized intake summary and search brief ready for sourcing kickoff.
5. **Action Tracking** — Tracks follow-up items, decisions made, and next steps from the meeting.

## Key Features

- Structured intake meeting templates and guided workflows
- Role requirement capture with priority weighting (must-have, nice-to-have, deal-breaker)
- Hiring manager alignment scoring and gap identification
- Auto-generated intake summaries and search briefs
- Action item tracking and follow-up reminders
- Integration-ready output for sourcing pipeline kickoff
- Dashboard view for managing multiple active searches
